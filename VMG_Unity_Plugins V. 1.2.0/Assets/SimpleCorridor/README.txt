Thanks for purchasing Simple Corridors Pack.

To make showcase scene look correctly, download and import Post Processing Stack to your project (https://www.assetstore.unity3d.com/en/#!/content/83912).
Door script can work with any player, which has "Player" tag. The door can be opened by pressing "E" key on the keyboard.
By default on the scene, you can find only static camera, but you can import first person character from Standard Assets, add Post Processing Behaviour Component to the FPC camera and get the same result, like in project video.
